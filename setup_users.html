⚠ Halaman ini hanya untuk setup awal. Hapus file ini dari hosting setelah selesai.
1
Jalankan SQL migrasi migrate_to_supabase_auth.sql di Supabase Dashboard → SQL Editor.
File ini membuat tabel profiles, trigger otomatis, dan RLS policy baru.
2
Buka Supabase Dashboard → Authentication → Users → Add user.
Isi email dan password admin, centang "Auto Confirm User", lalu klik Create user.
